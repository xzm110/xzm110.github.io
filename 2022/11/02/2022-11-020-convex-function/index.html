definiton of convex function

some methods to proof fff is convex

see 03_functions_newhyx.pdf (pku.edu.cn)

Assume dom f\text{dom }fdom f is convex in the following discussion.

  1. A function f(x):Rn→Rf(x): R^n \to Rf(x):Rn→R is convex iff g(t)=f(x+tv):R→Rg(t) = f(x + tv): R \to Rg(t)=f(x+tv):R→R is convex w.r.t. ttt and dom g={t∣x+tv∈dom f},∀x,v∈dom f\text{dom }g = \{t | x + tv \in \text{dom }f\}, \forall x, v \in \text{dom }fdom g={t∣x+tv∈dom f},∀x,v∈dom f.
  2. fff is convex iff epi f\text{epi } fepi f is convex. see Epigraph (mathematics) - Wikipedia.
  3. (first order derivative) Suppose fff is differentiable, then fff is convex if and ony if f(y)≥f(x)+∇f(x)T(y−x),∀x,y∈dom ff(y) \ge f(x) + \nabla f(x)^T (y-x) , \forall x, y \in \text{dom }ff(y)≥f(x)+∇f(x)T(y−x),∀x,y∈dom f.
  4. (first order derivative) Suppose fff is differentiable, then fff is convex if and only if ∇f(x)\nabla f(x)∇f(x) is monotone i.e. (∇f(y)−∇f(x))T(y−x)≥0,∀x,y∈dom f\left( \nabla f(y) - \nabla f(x) \right)^T (y-x) \ge 0, \forall x, y \in \text{dom }f(∇f(y)−∇f(x))T(y−x)≥0,∀x,y∈dom f.
  5. (second order derivative) Suppose all second-order partial derivatives of fff exist and are continuous, then fff is convex iff ∇2f(x)⪰0,∀x∈dom f\nabla^2 f(x) \succeq 0, \forall x \in \text{dom }f∇2f(x)⪰0,∀x∈dom f.
