definition of some sets in optimization

  1. convex set
    Set S is a convex set iff $\forall x_1, x_2 \in S, \theta x_1 + (1-\theta)x_2 \in S, \forall \theta \in (0,1)$.
  2. affine set
    Set S is an affine set iff $\forall x_1, x_2 \in S, \theta x_1 + (1-\theta)x_2 \in S, \forall \theta \in \mathbb{R}$.
  3. cone
    Set S in a cone iff $\forall x \in S, \forall \theta \ge 0, \theta x \in S$.

Remark:
$S \subset \mathbb{R}^n$.
