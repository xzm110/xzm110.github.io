definition of some sets in optimization

  1. convex set
    Set S is a convex set iff ∀x1,x2∈S,θx1+(1−θ)x2∈S,∀θ∈(0,1)\forall x_1, x_2 \in S, \theta x_1 + (1-\theta)x_2 \in S, \forall \theta \in (0,1)∀x1​,x2​∈S,θx1​+(1−θ)x2​∈S,∀θ∈(0,1).
  2. affine set
    Set S is an affine set iff ∀x1,x2∈S,θx1+(1−θ)x2∈S,∀θ∈R\forall x_1, x_2 \in S, \theta x_1 + (1-\theta)x_2 \in S, \forall \theta \in \mathbb{R}∀x1​,x2​∈S,θx1​+(1−θ)x2​∈S,∀θ∈R.
  3. cone
    Set S in a cone iff ∀x∈S,∀θ≥0,θx∈S\forall x \in S, \forall \theta \ge 0, \theta x \in S∀x∈S,∀θ≥0,θx∈S.

Remark:
S⊂RnS \subset \mathbb{R}^nS⊂Rn.
