For Markov chain, assume fii<1,fjj<1f _{ii} < 1, f _{jj} < 1fii​<1,fjj​<1,
proof ∑n=1∞Pijn<∞\sum_{n=1}^\infty P _{ij}^n < \infty∑n=1∞​Pijn​<∞,
and fij=∑n=1∞Pijn1+∑n=1∞Pjjnf _{ij} =\frac{\sum _{n=1}^\infty P _{ij}^n}{1 + \sum _{n=1}^\infty P _{jj}^n}fij​=1+∑n=1∞​Pjjn​∑n=1∞​Pijn​​

Note that

\begin{align} \sum_{n=1}^\infty {P_{ij}^n} &= \sum_{n=1}^\infty {\sum_{l=1}^n{f_{ij}^l {P_{jj}^{n-l}}}} \\\\ &= \sum_{l=1}^\infty { \sum_{n=l}^\infty {f_{ij}^l P_{jj}^{n-l} } } \\\\ &= \sum_{l=1}^\infty {\sum_{n=0}^\infty {f_{ij}^l {P_{jj}^{n}} } } \\\\ &= \sum_{l=1}^\infty {f_{ij}^l \sum_{n=0}^\infty { P_{jj}^{n}} } \\\\ &= f_{ij} \left( \sum_{n=0}^\infty P_{jj}^n \right) \\\\ &= f_{ij} \left( \frac1{1 - f_{jj}} \right) \\\\ &\le \frac1{1 - f_{jj} } \\\\ &< \infty \\\\ \end{align}

By the proof above, specifically the last equation
fij=∑n=1∞Pijn1+∑n=1∞Pjjnf_{ij} = \frac{\sum_{n=1}^\infty P_{ij}^n}{1 + \sum_{n=1}^\infty P_{jj}^n}fij​=1+∑n=1∞​Pjjn​∑n=1∞​Pijn​​
